# Fake News Detection
It is a basic ML-based project to predict whether a news article is Fake or Real.

## Dataset
Dataset not included due to size.

You can download:
- Fake.csv
- True.csv
(from Kaggle Fake News Dataset)

## Library and Module used
- Pandas
- NLTK
- Sci-kit learn
- Pickle
- re (Regular Expressions)

## Model uses
- TF-IDF Vectorization
- Logistic Regression

## Accuracy
- Achieved 98.95% accuracy on test data

## What I learned
- TF-IDF vectorization
- Data cleaning
- csv file handling using Pandas

# How to Run

### Train the model:

python train.py

### Run predictions:

python predict.py

Enter a news article when prompted to receive a prediction.
